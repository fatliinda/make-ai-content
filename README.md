# make-ai-content
AI workflow that turns business info into a 30-day social media content plan automatically.
Small businesses struggle to create social media content consistently. 
This Make.com automation generates a full 30-day social media content plan from a simple form submission.
Workflow:
Google Form → Google Sheets trigger → Google Gemini AI → Generated content output.
The system turns business details into ready-to-use post ideas, captions, hooks and hashtags in seconds.

Tech
Make.com
Google Forms
Google Sheets
Google Gemini API
